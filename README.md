# QuizApp
Run the project using QuizApp.py.
The Tests folder contains tests for the main functions in the respective directories.
The Quiz functionalities folder contains files that help in communicating with the database and storing questions.
The GUI functionalities folder contains the main class that manages the GUI and the application's logic.
The GUI folder only contains the generated UI file using QtDesigner.
The Data files folder includes two files: one with custom questions and the other with high score leaderboards.
